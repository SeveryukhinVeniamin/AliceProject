Команда: Северюхин Вениамин и Аникин Роман

Данный проект - это навык для Алисы, работающий с картами.
Помогает найти объекты, простроить маршруты и найти их длину в км.
Программа работает с помощью api-запросов к приложениям
geocode и static-maps. Работает с контентом пользователя.
Маршруты строит по заданным точкам.

Этапы разработки:
 1. Создание команды, репозитория и задумки.
 2. Создание ТЗ.
 3. Каркас программы.
 4. Вспомогательные функции.
 5. Черновик презентации.
 6. Программы выполняет основные функции. Отладка.
 7. Всё готово. Защита.
